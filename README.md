# 🌡️ SD Pro Temperature Converter

A feature-rich GUI-based Temperature Converter developed using Python and Tkinter as part of the SkillCraft Technology Software Development Internship.

## 📌 Project Overview

SD Pro Temperature Converter is more than a basic temperature conversion tool. It provides an interactive and user-friendly interface for converting temperatures between Celsius, Fahrenheit, and Kelvin while offering additional utilities such as graphical visualization, formula reference, conversion history tracking, PDF report generation, and clipboard support.

---

## ✨ Features

### 🔄 Temperature Conversion
- Celsius ↔ Fahrenheit
- Celsius ↔ Kelvin
- Fahrenheit ↔ Kelvin
- Instant and accurate conversions

### 📈 Graph Visualization
- Displays temperature values using graphical representation.
- Helps users analyze and compare converted values visually.

### 📋 Formula Sheet
- Built-in reference sheet containing all temperature conversion formulas.
- Useful for learning and verification purposes.

### 📜 Conversion History
- Maintains a history of all conversions performed during the session.
- Enables users to review previous calculations.

### 📄 PDF Report Generation
- Generate and save conversion results as PDF files.
- Useful for documentation and record keeping.

### 📋 Copy to Clipboard
- One-click copy functionality for converted results.
- Simplifies sharing and reusing values.

### 🖥️ Modern GUI
- User-friendly interface built with Python Tkinter.
- Clean layout with attractive styling.
- Input validation and error handling.

## 🧮 Temperature Conversion Formulae

### Celsius to Fahrenheit
```text
F = (C × 9/5) + 32
```

### Fahrenheit to Celsius
```text
C = (F - 32) × 5/9
```

### Celsius to Kelvin
```text
K = C + 273.15
```

### Kelvin to Celsius
```text
C = K - 273.15
```

### Fahrenheit to Kelvin
```text
K = (F - 32) × 5/9 + 273.15
```

### Kelvin to Fahrenheit
```text
F = (K - 273.15) × 9/5 + 32
```
